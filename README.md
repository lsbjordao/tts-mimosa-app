# TTS-Mimosa-App

The project TTS-Mimosa-App is a web application built with Next.js and TypeScript. It serves as a front-end interface for interacting with the TTS-Mimosa data structure, allowing users to view and interact with the taxonomic descriptions of organisms.

# TypeTaxonScript

The project [TTS-Mimosa](https://github.com/lsbjordao/TTS-Mimosa) is developed under the TypeTaxonScript principles, a new way to develop collaborative and well-documented taxonomic descriptions using TypeScript.
